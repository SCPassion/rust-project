# Rust Tutorial Project

This is a tutorial project for learning Rust programming concepts. Each lesson in the `src/lessons/` folder focuses on a specific Rust concept, making it easy to study and practice individual topics.

## Project Structure

```
src/
├── main.rs              # Main entry point
└── lessons/             # Individual concept lessons
    ├── ...                   # More lessons to come

```

## Lessons

Each file in the `lessons/` folder contains focused examples and explanations for a specific Rust concept

## Some basic rust commands in command line

```bash
# Create a new Rust project
cargo new rust-project
cd rust-project

# Build the project
cargo build

# Run the project
cargo run

# Check the code for errors without compiling
# This is useful to check the code for errors before compiling
cargo check

# Format the whole codebase
rustfmt src/**/*.rs
```
